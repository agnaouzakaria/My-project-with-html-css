# My Registration Page Project

## 📝 Description
Ce projet est une interface d'inscription interactive et responsive, développée avec **HTML5** et **CSS3**. C'est un projet personnel réalisé en **auto-formation** pour approfondir mes compétences en développement Front-end et en design d'interface (UI).

## 🚀 Technologies utilisées
* **HTML5** : Structure de la page et des formulaires.
* **CSS3** : Mise en page, stylisation avancée et design visuel.
* **GitHub Pages** : Hébergement du projet en ligne.

## 🛠️ Fonctionnalités
* Formulaire d'inscription complet (Nom, Email, Date de naissance, Mot de passe).
* Design moderne avec une thématique "Tech/Réseaux" cohérente.
* Interface adaptée aux différents types d'écrans (Responsive design).
